# Markdown Editor

一个 Mac 风格的 Markdown 编辑器，支持实时预览、代码高亮、数学公式、一键复制富文本等功能。

---

无需安装，直接访问：

**[https://poker-s.github.io/markdown_online/](https://poker-s.github.io/markdown_online/)**

## 本地使用

### 第一次使用（环境准备）

如果你的电脑从未运行过 Node.js 项目，需要先完成以下步骤。**已安装过的可以跳过。**

**第一步：安装 Node.js**

1. 访问 [https://nodejs.org](https://nodejs.org)，下载 **LTS（长期支持版）**
2. 双击安装包，一路点击「Next」
3. 安装完成后验证：

```bash
node -v
npm -v
```

两条命令都输出版本号即安装成功。

**第二步：安装项目依赖（只需做一次）**

在项目目录下打开终端，运行：

```bash
npm install
```

### 日常启动

```bash
npm run dev
```

启动后用浏览器打开 `http://localhost:5173` 即可使用。

> 关闭编辑器：在终端按 `Ctrl + C`。

> **npm install 很慢？** 可先切换国内镜像：
> ```bash
> npm config set registry https://registry.npmmirror.com
> ```

## 功能介绍

### 顶部按钮

| 按钮 | 功能 |
|------|------|
| Open | 打开本地 `.md` / `.txt` 文件 |
| Save | 保存为 `.md` 文件（快捷键 `Ctrl+S` / `Cmd+S`） |
| Export | 导出为 `.html` 网页文件 |
| Copy | 复制富文本到剪贴板，可直接粘贴到**微信公众号编辑器**，保留标题、代码块、表格、数学公式等样式 |
| Edit / Split / Preview | 切换「仅编辑 / 分栏 / 仅预览」三种视图 |
| ⚙️ | 图片上传配置 |
| ⊞ / ⊡ | 专注模式（隐藏工具栏） |
| 🌙 / ☀️ | 切换暗色 / 亮色主题 |

### 编辑器工具栏

- **文字格式**：加粗、斜体、删除线、行内代码
- **标题**：H1 / H2 / H3
- **列表**：无序列表、有序列表、任务列表
- **块元素**：引用、代码块、分割线
- **插入**：链接、图片、表格

### 代码高亮

````markdown
```javascript
console.log('Hello World')
```
````

### 数学公式（KaTeX）

- 行内公式：`$E = mc^2$`
- 块级公式：`$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$`

### 图片粘贴

直接将图片**复制粘贴**到编辑器，自动转换为 Markdown 图片语法。

> **当前限制：** 图片以 Base64 编码内联存储，会生成很长的数据链接。文档图片较多时文件体积会显著增大。
>
> **后续版本计划：** 支持配置图床（如 SM.MS、GitHub、自定义 OSS），将图片上传后替换为短链接。

## 常见问题

**Q：端口 5173 被占用怎么办？**

Vite 会自动尝试下一个可用端口（5174、5175…），按终端提示的地址打开即可。

**Q：关掉终端后还能用吗？**

不能，必须保持终端运行。建议最小化终端窗口，不要关闭。
