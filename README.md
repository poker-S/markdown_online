# Markdown Editor

一个 Mac 风格的 Markdown 编辑器，支持实时预览、**可视化插入**、代码高亮、数学公式、一键复制富文本等功能。

## 本地使用

### 第一次使用（环境准备）

如果你的电脑从未运行过 Node.js 项目，需要先完成以下步骤。**已安装过的可以跳过。**

**第一步：安装 Node.js**

1. 访问 [https://nodejs.org](https://nodejs.org)，下载 **LTS（长期支持版）**
2. 双击安装包，一路点击「Next」
3. 安装完成后验证：

```bash
node -v
npm -v
```

两条命令都输出版本号即安装成功。

**第二步：安装项目依赖（只需做一次）**

在项目目录下打开终端，运行：

```bash
npm install
```

### 日常启动

```bash
npm run dev
```

启动后用浏览器打开 `http://localhost:5173` 即可使用。

> 关闭编辑器：在终端按 `Ctrl + C`。

> **npm install 很慢？** 可先切换国内镜像：
> ```bash
> npm config set registry https://registry.npmmirror.com
> ```

### 编辑器工具栏

- **文字格式**：加粗、斜体、删除线、行内代码
- **标题**：H1 / H2 / H3
- **列表**：无序列表、有序列表、任务列表

- **✦ 可视化插入**：Word 风格所见即所得编辑器，详见下方说明

## ✦ 可视化插入

### 为什么需要它？

Markdown 的语法对于技术用户来说直观高效，但对于**不熟悉 Markdown 语法的用户**来说，记住 `**加粗**`、`## 标题`、`| 表格 |` 这些符号有一定门槛。

可视化插入解决了这个问题：**不需要记任何语法，像用 Word 一样写内容，最终自动转换为标准 Markdown 格式**。这让那些只是偶尔需要写 Markdown 文档的用户，也能流畅完成工作。

### 使用方式

1. 点击工具栏 **✦ 可视化插入** 按钮
2. 首屏展示动态标题动画，点击「开始使用 →」进入编辑器
3. 在编辑区自由输入内容
4. **选中文字后点击上方格式按钮**，即可应用对应样式（与 Word 操作完全一致）
5. 点击右下角 **确认插入 ✓**，内容自动转为 Markdown 并插入到编辑器光标所在位置

### 支持的格式操作

| 按钮 | 效果 | 转换结果 |
|------|------|----------|
| H1 / H2 / H3 | 标题样式 | `# 标题` |
| B | 加粗 | `**文字**` |
| I | 斜体 | `*文字*` |
| S | 删除线 | `~~文字~~` |
| ` | 行内代码 | `` `代码` `` |
| ❝ | 引用块 | `> 引用` |
| • 列表 | 无序列表 | `- 项目` |
| 1. 列表 | 有序列表 | `1. 项目` |
| 🔗 链接 | 超链接 | `[文字](url)` |
| 🔲 表格 | 网格选择行列数，插入可编辑表格 | 标准 Markdown 表格 |
| 🖼 图片URL | 输入图片地址插入图片 | `![alt](url)` |
| 📁 上传图片 | 从本地选择图片文件，自动转 Base64 | `![name](data:...)` |

**粘贴截图**：在可视化编辑器内直接 Ctrl+V 粘贴截图，自动转 Base64 插入。

### 适合哪些场景？

- 不熟悉 Markdown 语法，但需要输出 Markdown 格式文档
- 快速插入复杂结构（如多列表格），不想手写语法
- 从 Word / 富文本内容迁移到 Markdown 工作流的过渡期

## 使用指南面板

点击顶栏「Guide」按钮，从右侧滑入使用指南面板，包含两部分内容：

- **Markdown 语法速查**：标题、文字格式、列表、链接与图片、引用、代码块、表格、数学公式，每条附有语法示例和说明
- **编辑器功能说明**：工具栏、表格构建器、公式库、图片粘贴、可视化插入、Copy 富文本、Export HTML、分栏拖拽、专注模式、快捷键、自动保存、Reset

面板内容可自由选中复制。点击遮罩或右上角 ✕ 关闭。

## 自动保存与 Reset

内容、文件名、主题变化后自动保存到浏览器本地存储，刷新页面后自动恢复。状态栏左侧显示最后保存时间。

点击顶栏 **Reset** 按钮可清空所有本地数据（内容、文件名、主题、图片设置），恢复到初始状态。操作前会弹出确认框，确认后不可撤销。

## 图片粘贴

直接将图片**复制粘贴**到编辑器，自动转换为 Markdown 图片语法。粘贴后 Base64 长串在编辑器中自动折叠为一行显示，鼠标悬浮可查看数据信息，右键可展开/收起。

首次使用时会弹出图片存储方式说明，可在 ⚙️ 设置中随时切换：

| 方案 | 说明 |
|------|------|
| 内嵌 Base64（默认） | 图片嵌入文件，可在任何地方打开，文件体积较大 |
| 本地文件引用 | 图片以相对路径引用，文件体积小，需与 .md 文件放在同一目录，在线版不支持 |

## 常见问题

**Q：端口 5173 被占用怎么办？**

Vite 会自动尝试下一个可用端口（5174、5175…），按终端提示的地址打开即可。

**Q：关掉终端后还能用吗？**

不能，必须保持终端运行。建议最小化终端窗口，不要关闭。
