# Markdown Editor

一个 Mac 风格的 Markdown 编辑器，支持实时预览、图片粘贴、代码高亮、数学公式等功能。

---

## 第一次使用（环境准备）

如果你的电脑从未运行过 Node.js 项目，需要先完成以下步骤。**已安装过的可以跳过。**

### 第一步：安装 Node.js

1. 打开浏览器，访问 [https://nodejs.org](https://nodejs.org)
2. 点击左边绿色的 **LTS（长期支持版）** 按钮下载
3. 下载完成后双击安装包，一路点击「Next」即可
4. 安装完成后，打开终端（Windows 按 `Win + R` 输入 `cmd` 回车），输入以下命令验证是否安装成功：

```bash
node -v
npm -v
```

如果两条命令都输出了版本号（例如 `v20.x.x`），说明安装成功。

---

### 第二步：安装项目依赖（只需做一次）

打开终端，进入项目所在的文件夹。

**方法一（推荐）：** 在文件资源管理器中，打开 `markdown-editor` 文件夹，在地址栏输入 `cmd` 回车，终端会自动定位到该目录。

**方法二：** 手动输入路径，将下面的路径替换为你实际存放项目的位置：

```bash
cd "你的项目路径/markdown-editor"
# 例如：cd "C:/Users/你的用户名/Desktop/markdown-editor"
```

进入目录后，运行：

```bash
npm install
```

等待安装完成（可能需要 1~2 分钟，取决于网速）。

---

## 日常启动

每次使用时，用同样的方式打开终端并进入项目目录，然后运行：

```bash
npm run dev
```

启动成功后，终端会显示类似：

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

用浏览器打开 `http://localhost:5173` 即可使用。

> 关闭编辑器：在终端按 `Ctrl + C` 停止服务。

## 功能介绍

### 编辑与预览
| 功能 | 说明 |
|------|------|
| 分栏预览 | 左侧写 Markdown，右侧实时渲染效果 |
| 拖拽分割线 | 鼠标拖动中间分割线，自由调整左右宽度 |
| 三种视图 | 顶部切换「Edit / Split / Preview」三种模式 |
| 专注模式 | 点击 ⊞ 按钮隐藏工具栏，沉浸写作 |

### 图片支持
直接将图片**复制粘贴**到编辑器，会自动转换为 Markdown 图片语法并嵌入文档，无需上传服务器。

### 工具栏
- **文字格式**：加粗、斜体、删除线、行内代码
- **标题**：H1 / H2 / H3
- **列表**：无序列表、有序列表、任务列表
- **块元素**：引用、代码块、分割线
- **插入**：链接、图片、表格

### 代码高亮
代码块支持语法高亮，例如：

````markdown
```javascript
console.log('Hello World')
```
````

### 数学公式（KaTeX）
- 行内公式：`$E = mc^2$`
- 块级公式：`$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$`

### 文件操作
| 操作 | 方式 |
|------|------|
| 打开文件 | 点击顶部「Open」按钮，支持 `.md` / `.txt` |
| 保存文件 | 点击「Save」或按 `Ctrl+S`（Mac 用 `Cmd+S`） |
| 导出 HTML | 点击「Export」导出为网页文件 |

### 主题切换
点击右上角 🌙 / ☀️ 按钮切换暗色 / 亮色主题。

## 常见问题

**Q：`npm install` 很慢怎么办？**

可以切换为国内镜像源再安装：

```bash
npm config set registry https://registry.npmmirror.com
npm install
```

**Q：端口 5173 被占用怎么办？**

Vite 会自动尝试下一个可用端口（5174、5175…），终端会提示实际地址，按提示打开即可。

**Q：关掉终端后还能用吗？**

不能，必须保持终端运行。建议最小化终端窗口，不要关闭。
